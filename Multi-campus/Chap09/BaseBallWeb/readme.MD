EOSJS 환경구성

### 1. nodeJS 설치
> sudo apt update
>
> sudo apt install node

### 2. npm 설치
> sudo apt install npm

### 3. create-react-app 설치
> sudo npm install -g create-react-app

### 4. react test app 실행
> create-react-app test

### 5. EOSJS 설치
> cd test
>
> npm install eosjs@16.0.9
